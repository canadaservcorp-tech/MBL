## English Installation Guide

### Accessing the Application

**Web Access (Recommended)**
1. Open your web browser (Chrome, Firefox, Safari)
2. Go to: `https://your-app-url.netlify.app`
3. Bookmark this page for easy access

**Login Credentials**
- Your administrator will provide your:
  - Email address
  - Temporary password
- Change your password after first login

---

### First Time Login

1. **Enter Your Email**
   - Use the email provided by your administrator
   
2. **Enter Your Password**
   - Use the temporary password provided
   
3. **Click "Sign In"**

4. **Important:** Change your password in Settings after first login

---

### Main Features

#### Dashboard
- View task summary and statistics
- See upcoming tasks
- Quick overview of maintenance status

#### Tasks
- View all assigned tasks
- Update task status (Pending → In Progress → Completed)
- Add remarks and actual costs
- Filter by status, priority, or date

#### Preventive Maintenance
- View scheduled preventive tasks
- Check upcoming maintenance dates
- Review maintenance history

#### Contractors
- Access contractor contact information
- View contractor ratings and specialties
- Phone and email details

#### Apartments
- View all 36 apartments
- See common areas and service spaces
- Access apartment-specific maintenance history

---

### Using the Task System

#### Viewing Your Tasks
1. Click "Tasks" in the left menu
2. Your assigned tasks appear in the list
3. Use filters to find specific tasks

#### Updating a Task
1. Click the Edit icon (✏️) next to the task
2. Update the status:
   - **Pending:** Not started
   - **In Progress:** Currently working on it
   - **Completed:** Finished
   - **Cancelled:** No longer needed
3. Add remarks if needed
4. Enter actual cost if different from estimate
5. Click "Update Task"

#### Adding Remarks
- Click Edit on any task
- Scroll to "Remarks" field
- Add notes about the work performed
- Include any issues encountered
- Click "Update Task" to save

---

### Tips for Best Results

✅ **Update task status regularly** - Keep everyone informed  
✅ **Add detailed remarks** - Help future maintenance  
✅ **Report actual costs** - Improve budget accuracy  
✅ **Check preventive schedule** - Stay ahead of issues  
✅ **Use priority indicators** - Focus on urgent tasks first

### Utilisation du Système de Tâches

#### Consulter Vos Tâches
1. Cliquez sur "Tâches" dans le menu de gauche
2. Vos tâches assignées apparaissent dans la liste
3. Utilisez les filtres pour trouver des tâches spécifiques

#### Mettre à Jour une Tâche
1. Cliquez sur l'icône Modifier (✏️) à côté de la tâche
2. Mettez à jour le statut :
   - **En attente :** Non démarré
   - **En cours :** Travail en cours
   - **Complété :** Terminé
   - **Annulé :** Plus nécessaire
3. Ajoutez des remarques si nécessaire
4. Entrez le coût réel si différent de l'estimation
5. Cliquez sur "Mettre à Jour la Tâche"

#### Ajouter des Remarques
- Cliquez sur Modifier pour n'importe quelle tâche
- Faites défiler jusqu'au champ "Remarques"
- Ajoutez des notes sur le travail effectué
- Incluez tout problème rencontré
- Cliquez sur "Mettre à Jour la Tâche" pour sauvegarder
